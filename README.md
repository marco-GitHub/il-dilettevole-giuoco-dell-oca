# il-dilettevole-giuoco-dell-oca
Goose game java implementation

### Prerequisites

java 8

### Clone

Clone this repo to your local machine using `https://github.com/marco-GitHub/il-dilettevole-giuoco-dell-oca`

## Running

```
$ ./mvnw test
```

or

```
$ ./mvnw.cmd test
```
